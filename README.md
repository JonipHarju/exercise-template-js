## Getting Started

1. **Clone this repository and navigate into the folder:**

   ```bash
   git clone https://github.com/JonipHarju/exercise-template-js.git
   cd exercise-template-js
   ```

2. **Install dependencies:**
   ```
   npm install
   ```
3. **Do the exercises in "exercises.js" file.**

4. **To run the tests:**
   ```
   npm test
   ```
